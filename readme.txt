PROJECT: GLS Infrastructure Planning Portfolio — "Living CV"
Purpose
A real-time 3D visualisation of Stellenbosch's municipal water infrastructure, built as an unsolicited portfolio piece targeting the folks at GLS, a consulting company based in Stellenbosch. This is not a response to an advertised position — it is a deliberate introduction. The author is Hennie Kotze, a seasoned developer / GIS enthusiast based in Stellenbosch.
